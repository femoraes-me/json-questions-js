## 📖 Sobre 

Neste projeto foi feito um Quiz dinâmico utilizando JavaScript para montagem das questões na tela, para armazenamento das questões foi utilizado uma variável em um outro arquivo com um array de objetos, atividade feita como exercício proposto no curso de JavaScript da b7web do professor Bonieky Lacerda.

---

## 🚀 Tecnologias utilizadas
O projeto está sendo desenvolvido com as seguintes tecnologias:
- HTML5
- CSS3
- JavaScript

---

## ⌨ Como executar o projeto

Baixe o arquivo .ZIP do projeto e após descompactar abra com seu navegador o arquivo index.html.
